Ce este un website



Un site web este o colecție de pagini web accesibile public, interconectate, care au un singur nume de domeniu. Site-urile web pot fi create și întreținute de către o persoană, un grup, o afacere sau o organizație pentru a servi mai multor scopuri. Împreună, toate site-urile web accesibile public constituie World Wide Web.Deși uneori este numită „pagină web”, această definiție este greșită, deoarece un site web este format din mai multe pagini web. Un site web este cunoscut și ca „prezență web” sau pur și simplu „site”.

Landing Page



Landing page se poate traduce ca "pagină de destinație" - prima pagină pe care „aterizează” un utilizator din exterior, atras pe site în cadrul anumitor acțiuni de marketing. Însă pentru a înțelege mai bine ce este un landing page este important să descriem modul în care arată acest tip de subpagini și scopul acestia. Scopul principal al unui landing page este de a convinge utilizatorii să facă o anumită acțiune pe pagina dvs. web.

Magazin Online



Un magazin online sau magazin electronic (e-shop), este un website de comerț electronic destinat vânzării de produse și servicii. În cele mai multe dintre cazuri, magazinul online reprezintă o platformă pe care sunt adăugate produse. Aceste produse pot fi accesate și cumpărate cu ajutorul cărții de credit. Pe magazinele online complexe utilizatorii pot selecta modalitatea de plată: card de credit, transfer bancar, ramburs și modalitatea de transport: poștă, curier, avion etc.

Site de Prezentare



Este o posibilitate de a comunica eficient cu publicul țintă. Acesta conține informațiile-cheie despre companie și produsele ei – argumente exacte ce descriu avantajele unui produs sau serviciu, datele de contact și lista de prețuri.

Site Catalog



Pentru a prezenta produsele unui public larg, dar fără a cheltui mult pentru dezvoltarea unui magazin online, un site de catalog la cheie este soluția potrivită.

Design Web



Prin design web se înțelege în general realizarea de site-uri web, de la momentul conceperii structurii și interfeței grafice și până la finalizarea programării și introducerea propriu-zisă a datelor care alcătuiesc conținutul site-ului - imagini, text, fișiere și alte elemente.
